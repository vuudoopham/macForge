# Manual steps for this machine

These items were detected during the last backup and are not fully automatable.

## Common items that require manual action
- Log into LastPass (early)
- Log into ExpressVPN / Tailscale
- Enter license keys for Moom, Parallels, CrossOver, CleanMyMac, etc.
- Sign into Steam / Battle.net and download games
- Sign into Microsoft 365 apps
- First-run configuration for many apps

Run `./macForge apply --machine <this-machine>` to see the live list.
